Informasi Server Config

Halaman Server Config digunakan untuk mengatur konfigurasi server aplikasi ASIA PAYMENT.

Saat ini halaman ini hanya berisi informasi. Nantinya di sini bisa ditambahkan form untuk:

  • Mengatur API Base URL (endpoint OtomaX atau server lainnya).
  • Menyimpan Firebase Config secara dinamis.
  • Mengelola koneksi terminal server.
  • Melihat status server (online / offline).

Jika halaman ini belum dipakai, biarkan saja sebagai dokumentasi. Saat diperlukan, bisa ditambahkan form input agar admin dapat mengganti konfigurasi server tanpa mengubah kode sumber.
